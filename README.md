# create-tsl

**Note:** This will be a preset for [create-idea](https://github.com/Le0X8/create-idea) and this package will just be a wrapper in the future.

## Installation

```bash
npm create tsl@latest
```

## Features

### Workspace

- `mkdir`
- `git init` (optional)

### Node

- create `package.json`
- create `tsconfig.json`
- setup tsup
- setup jest
- setup eslint
- setup prettier
